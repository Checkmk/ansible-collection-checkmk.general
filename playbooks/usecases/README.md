# Use Cases Example Playbooks

This directory contains Ansible playbooks that demonstrate specific, real-world use cases and advanced scenarios for automating Checkmk.

## Contents

These playbooks go beyond basic module usage and show how to solve particular problems or implement specific workflows.

## Usage

You can run these playbooks similarly to the demo playbooks, ensuring you have the necessary inventory and variable configurations in place.

```bash
ansible-playbook -i playbooks/inventory/hosts.ini lookup-version.yml
```
